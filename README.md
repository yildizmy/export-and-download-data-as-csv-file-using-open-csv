## Getting Started

### Dependencies

* Spring Web
* Spring Boot
* Spring Data JPA
* OpenCSV
* H2 Database


### Installing

* Database connection url for DBeaver, etc.

```
jdbc:h2:tcp://localhost:9092/mem:employee-db
```

<br/>

### Executing program

* Postman request:

```
http://localhost:8080/api/v1/employees/export
```
